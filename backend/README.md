# Backend Server - Test Simulator

This backend runs a Node.js server that controls the Python-based test simulator, manages file uploads, and streams real-time input/output to the frontend.

## Requirements

- Node.js (v14+)
- Python 3 installed and available in your system PATH

## How to Run

1. Clone the repository and navigate to the backend directory.
2. Run the following command to install dependencies and start the server:

```bash
npm install && node server.js
```
